# LINE 貼圖 4×4 咒語產生器 - UI 簡化與流程優化方案

## 📊 當前狀況分析

### 現有複雜度
- **代碼行數**: 5,449 行
- **互動元素**: 284 個（section, button, input, select）
- **版本**: v2.0.9
- **功能**: 角色模板、快速套餐、腳本包、畫風、字體、批次輸出等

### 主要問題
1. ❌ **資訊過載**: 太多選項同時呈現
2. ❌ **流程不清晰**: 使用者不知道從哪裡開始
3. ❌ **學習曲線陡峭**: 新手需要時間理解所有功能
4. ❌ **視覺層次混亂**: 重要功能不突出

## 🎨 UI 簡化具體改進

### 1. 頂部導航簡化
**現在**: 多個按鈕、複雜的 preset dock
**改為**: 
```html
<div class="simple-nav">
  <button class="mode-btn active">快速模式</button>
  <button class="mode-btn">進階模式</button>
  <button class="help-btn">❓ 使用說明</button>
</div>
```

### 2. 摺疊式進階選項
**現在**: 所有選項平鋪展開
**改為**:
```html
<details class="advanced-section">
  <summary>🔧 進階設定（可選）</summary>
  <div class="advanced-content">
    <!-- 進階選項 -->
  </div>
</details>
```

### 3. 視覺層次優化
**改進**:
- 主要操作按鈕：大、醒目、藍色
- 次要操作：小、灰色
- 說明文字：更小、更淡

### 4. 減少同時顯示的元素
**策略**:
- 使用步驟式導航
- 一次只顯示一個區塊
- 完成後才顯示下一步
