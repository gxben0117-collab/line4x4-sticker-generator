# V2.60

Date: 2026-07-06

## 本輪主題

V3-lite 分類重整：把「句子資料庫」改成「使用者真的找得到句子的貼圖腳本挑選器」，版本升至 v2.6.0。

## 問題背景

1396 句（去重後）但使用者總覺得「不夠」——真因是找不到：86 個分類全部平權、最常用的快速回覆藏在深處、分類是資料整理思維不是 LINE 使用情境思維。

## 本輪完成

### 分類重整（規劃：docs/line-sticker-taxonomy-v3.md）

- 86 個舊分類收納進 **12 主分類 × 51 子分類**（舊分類名保留為 tags，資訊不遺失）。
- 新增 **🔥 熱門100**：跨分類虛擬入口，剛好 100 句 LINE 最高頻句子，**預設打開**。
- 熱門句在所有清單中排最前，並帶 🔥 標記。
- 語錄成語、渣男渣女、修仙、美髮沙龍等降權到「🎨 特殊主題」，排最後。
- 「全部 1396」按鈕改為「全部句庫」移到主分類列最後，不再是第一眼入口。

### UI 流程優化

- 新增**句子搜尋框**：直接輸入關鍵字搜全部句庫，熱門句優先排序；切分類自動清空搜尋。
- 主分類 chips 帶 emoji 加粗；單一子分類的主題（熱門100、寵物）自動隱藏子分類列，少一層點擊。
- 「隨機選擇本分類」改名「從目前清單隨機選」（涵蓋搜尋結果、熱門100 等情境）。
- workspace 還原後同步分類列渲染（修掉還原不生效的舊缺陷）；舊存檔的分類名自動對應到新子分類。
