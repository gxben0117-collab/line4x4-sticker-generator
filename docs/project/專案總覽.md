# 貼圖line4x4咒語產生器

## 專案定位

這個專案是面向 LINE 貼圖題材的靜態咒語產生器，重點是：

- 用單頁方式快速組合貼圖需求
- 保留大量可重用腳本與風格選項
- 保留快速套餐與一鍵套用所有模組設定
- 提供角色卡模板群組、批次輸出包與工作台保存
- 讓使用者可直接把結果拿去丟給 ChatGPT 或其他圖像流程

## 專案結構

- `index.html`
  使用者平常直接開啟的正式入口。

- `assets/`
  保留拆開的資料、樣式、腳本原稿，方便未來逐步模組化。

- `versions/`
  歷史 HTML 快照，不覆蓋舊版。

- `docs/`
  文件、版本規則、開發紀錄、handoff、程式邏輯。

- `scripts/build.mjs`
  基本建置腳本，將正式頁面輸出到 `dist/index.html`。

- `tests/smoke.test.mjs`
  最小健康檢查，確保版本、快照、build 輸出、模板群組、批次輸出、腳本修正區與快速套餐區存在。

- `scripts/lint.mjs`
  靜態片段檢查，確認 `v2.0.7` 重要節點與函式沒有意外消失。
