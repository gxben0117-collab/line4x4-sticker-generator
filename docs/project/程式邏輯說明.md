# 貼圖line4x4咒語產生器 程式邏輯說明

## 核心結構

目前專案仍以單一 `index.html` 為主，但實際運作邏輯可分成幾個模組：

- 狀態管理：`state`
- 預設套用：`presets` 與相關切換函式
- 角色卡模板：`templateGroups`、`characterTemplates`
- 腳本資料：`scriptData`、`scriptGroups`
- 腳本快速組合：`scriptQuickCombos`
- 腳本編輯：`toggleScript()`、`syncScriptFromTextarea()`、`fillScriptToSlots()`、`dedupeScriptEditor()`、`sortScriptEditor()`、`blendToneScriptEditor()`、`rebuildBalancedPack()`
- 腳本檢查：`getScriptHealthReport()`、`getFixSuggestions()`
- 批次輸出：`buildPromptFromFormat()`、`buildBatchOutputs()`、`renderBatchOutputPanel()`
- 最終組合：`combineAll()`
- 複製輸出：`copyFinal()`、`copyToClipboard()`
- 工作台保存：`persistWorkspace()`、`restoreWorkspace()`

## 腳本區邏輯

腳本區目前有兩條並行路徑：

1. 左側用分類與群組選取現成腳本
2. 右側直接把腳本整理成最終輸出內容

為了避免左右不同步，右側 `textarea` 每次輸入都會透過 `syncScriptFromTextarea()` 回寫到狀態。

## 腳本工具

目前右側腳本編輯區已有以下工具：

- `copyScriptEditor()`
  直接複製目前腳本編輯區文字。

- `clearScriptEditor()`
  清空目前腳本編輯區與已選腳本狀態。

- `fillScriptToSlots()`
  當前腳本不足格數時，自動從目前可見腳本池補滿。

- `dedupeScriptEditor()`
  去除重複台詞，避免同一句重複佔格。

- `sortScriptEditor()`
  依文字順序整理腳本，方便人工巡檢與比對。

- `blendToneScriptEditor()`
  自動補入不同語氣桶，避免全包都像同一種情緒。

- `rebuildBalancedPack()`
  依日常 / 回覆 / 陪伴 / 熱血 / 節慶重新組成平衡 16 句包。

## 角色卡模板與推薦搭配

`v2.0.7` 新增五大模板群組：

- 日常主角
- 工作職場
- 療癒陪伴
- 節慶歡樂
- 衝刺戰鬥

每個模板會附：

- 角色描述
- 服裝設定
- 圖片備註
- 建議模組快捷包
- 建議腳本快速組合

套用模板時，不會移除原本快速套餐與模組操作，而是在現有流程上加速導入。

## 批次輸出邏輯

新版保留單一輸出習慣，同時新增輸出包概念：

- `完整 Prompt`
- `精簡 Prompt`
- `JSON`

其中 JSON 會固定輸出：

- `qty`
- `source`
- `mode`
- `format`
- `character`
- `outfit`
- `imageNotes`
- `styleSettings`
- `lines`

## 工作台狀態與保存

首頁新增工作台摘要與模板群組摘要，會根據目前選擇顯示：

- 目前模板群組
- 輸出包數量
- 腳本健康度

工作台資料會用 localStorage 保存，重新開啟頁面後自動還原。

## 組合輸出邏輯

`combineAll()` 會把目前所有選項整合成最終 prompt，包含：

- 貼圖格數
- 角色描述
- 比例與風格
- 服裝
- 色系
- 字體
- 構圖
- 背景
- 美顏限制
- 腳本文字

## 維護原則

- 每次正式功能調整要升 HTML 版本
- 每次升版前先保留 `versions/` 快照
- 每次正式變更補 `development-log` 與 `handoff`
- 測試至少要覆蓋：
  - 主頁版本
  - 歷史版本保留
  - build 輸出存在
  - 新增功能/分類存在
  - 批次輸出與腳本修正區存在
