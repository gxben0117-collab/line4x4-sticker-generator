# 版本規則

## HTML 版本號

目前最新版本：

```text
v2.0.8  （index.html 正式版）
```

下次正式 HTML 版本建議接：

```text
v2.0.9
```

每次正式更新：
1. 先把目前 `index.html` 複製到 `versions/貼圖line4x4咒語產生器-vX.Y.Z.html`
2. 在 `index.html` 完成修改
3. 更新頁面內的版本標籤（`<title>`、`.version-badge`）
4. 更新 `scripts/lint.mjs` 裡的版本字串
5. 更新 `tests/smoke.test.mjs` 裡的版本字串
6. 執行 `npm run build && npm run lint && npm test`

## 文件版號

目前文件版號：`V2.08`

每次正式文件更新往後加 `0.01`（V2.08 → V2.09）。

新增文件時建立：
- `docs/development-log/VX.YY.md`
- `docs/project/VX.YY_HANDOFF.md`
