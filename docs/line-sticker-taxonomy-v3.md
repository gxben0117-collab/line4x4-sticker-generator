## 為什麼要改

目前 1396 句（去重後）不是不夠，是**找不到**：

1. 86 個分類全部平權，`三字經名句` 和 `收到` 同一層級。
2. 最常用的快速回覆（收到、好喔、早安、晚安）藏在多個分類裡，沒有統一入口。
3. 分類是「資料整理思維」（按來源批次分），不是「LINE 使用情境思維」（我現在想回什麼）。
4. 同一句重複出現在多個分類（1518 → 1396，重複 122 句次），UI 上看起來雜。

V3-lite 目標：**12 個主分類 + 約 42 個子分類 + 熱門度星等**，首頁預設打開 `🔥 熱門100句`。

## 熱門度星等規則

- `★★★★★`：熱門100句名單內（見下方）。
- `★★★★`：快速回覆、問候、早安晚安、職場回覆、通用祝福、生日等高頻子分類預設值。
- `★★★`：情緒、迷因、戀愛撒嬌、寵物、吃喝、天氣等中頻。
- `★★`：興趣向（追星、購物、讀書、失戀、玄學…）。
- `★`：特殊主題全區（語錄、玩梗、職業專用）。

排序規則：分類內按星等降冪，同星等保持原順序。
